## Descrizione
Applicazione Java per simulare la lettura e la scrittura di un file in modalità concorrente.

## Obiettivi
- Diagramma degli stati di un thread e transizioni
- Variabili di classe e thread safety
- Realizzazione di un thread user-level
- Gestione stream
- Gestione eccezioni
- Documentazione del codice in Java
- Diagramma UML delle classi e dei casi d’uso

## Osservazioni
Il progetto è stato scritto per scopi didattici e ha solo lo scopo di simulare una comunicazione Lettore Scrittore, di far comprendere le proprietà degli stream e delle eccezioni.
